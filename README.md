### ✔️ 세부 역할 분담

#### 🔹Backend

- 권대현: 복지 예약 / 카드 내역 / 이상 거래 탐지

- 정민교: PM / 똑똑이 / 대화방, 대화 내역 / CICD

- 양승건: 카드발급 / 알림 / 소비 리포트

- 홍정민: 로그인 / 회원가입 / 매칭 / CICD

#### 🔹Frontend

- 이두리: 메인 / 소비 내역 / 알림 페이지 / 로그인 / 회원가입

- 손동희: 복지 / 소비 리포트 / 생체 로그인

- 정성진: 카드 발급 / 매칭 / 똑똑이 / 메인
